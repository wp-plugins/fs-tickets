== Description ==

The FireStorm Support Ticket System is an simple, easy to use, support ticket system allowing members to easily create support tickets on your website. Tickets can be managed via the WordPress dashboard or on the front-end of your website. Ticket status and time is also integrated keeping both admins and users informed. Tickets can be categorized for easy management.

The FireStorm Support Ticket System works independent from blog comments and posts. This helps keep your blog and posts available for other use.

== Installation ==

1. Move the plugin directory to your wp-content/plugins directory.
2. Activate the plugin in the 'Plugins' section in your WordPress administration. The plugin will automatically create the needed WordPress Page.
